# Monstera Form Variation

> **Status: Snapshot / Not actively maintained** · Last substantive update 2026-03-05
>
> This repository is a static white paper artifact proposing a multi-axis character
> framework for replacing informal "long form" / "short form" terminology in Monstera
> with structured morphological and molecular evidence. It is not a software project
> and is not under active development. The paper is hosted as a single `index.html`
> file (also served via GitHub Pages). Open issues may go unanswered.

## What this is

A taxonomic clarification document addressing the informal "long form" vs "short form"
designations commonly used in the Monstera aroid community. The paper proposes a
multi-axis character framework grounded in morphological and molecular evidence,
rather than a single binary distinction.
